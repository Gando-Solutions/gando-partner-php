# Connect

## Overview

Build **signed Partner Connect** URLs so rental management software can send rental operators to the Gando dashboard to complete signup. After signup, your integration receives the Gando `account_id` via the **`rental_operator.linked`** webhook (see [Webhooks](../webhooks/README.md)).

This helper is **not** generated by Speakeasy. It lives in `Gando\Partner\Connect\UrlBuilder` ([`src/Connect/UrlBuilder.php`](../../../src/Connect/UrlBuilder.php)) and uses your **connect secret** (`gando_cs_…`), not the partner API key (`gando_pk_…`). Do **not** use `Gando\Partner\Api\Client` or `Gando\Partner\Gando` to build these URLs.

Typical flow:

1. Build a signup URL with `signupUrl` and redirect the rental operator (or return the URL to your frontend).
2. Subscribe to **`rental_operator.linked`** on your partner webhook endpoint.
3. Persist `account_id` and your `external_id` for subsequent Partner API calls (e.g. `POST /api/partner/deposits`).

See also `recipes/snippets/connect.signup-url.php`.

### Available Operations

* [signupUrl](#signupurl) - Build a signed rental-operator signup URL (`/register`)

## signupUrl

Returns a full HTTPS URL to `{baseUrl}/register` with query parameters `partner`, `external_id`, `ts`, and `sig` (HMAC-SHA256 hex over `{partnerSlug}.{externalId}.{ts}`). Optional query keys: `email`, `name`, `return_url`.

Use a fresh timestamp for each production link. Gando validates `sig` and `ts` server-side.

### Example Usage

<!-- UsageSnippet language="php" operationID="connect.signupUrl" method="get" path="/register" -->
```php
declare(strict_types=1);

require 'vendor/autoload.php';

use Gando\Partner\Connect\UrlBuilder;

$builder = new UrlBuilder(
    connectSecret: getenv('GANDO_CONNECT_SECRET') ?: '<YOUR_CONNECT_SECRET_HERE>',
    partnerSlug: 'fleetee',
    baseUrl: getenv('GANDO_DASHBOARD_BASE_URL') ?: 'https://dashboard.gando.app',
);



$signupUrl = $builder->signupUrl(
    externalId: 'fleet_acct_42',
    email: 'ops@example.com',
    name: 'Fleetee Ops',
    returnUrl: 'https://partner.example/connect/return',
);

if ($signupUrl !== '') {
    // handle response
}
```

### Response

**`string`** — Full signup URL including scheme, host, path `/register`, and query string.

### Errors

This helper does not call the Partner API. It does not return HTTP status codes or throw `Errors\ErrorEnvelope` / `Errors\APIException`. Invalid or expired signatures are rejected by the dashboard when the rental operator opens the URL.
